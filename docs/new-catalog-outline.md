## Content rules

- Use professional B2B English and avoid unsupported superlatives.
- Label standard values as “typical” or “common” where they are not guaranteed.
- Use “custom by review” instead of implying every specification is always available.
- Do not publish fake certificates, test values, customer logos or compliance marks.
- Distinguish raw-fiber properties from dry-fabric, cured-laminate and finished-profile properties.
- Add a final verification note to internal working files for company claims and contact data.
- Keep image frames replaceable without changing page layout.

6. Process / quality / packing page.

## Print and digital behavior

- Each `.catalog-page` is one A4 page.
- Browser view uses a dark workspace with page shadows and optional page navigation.
- Print mode removes navigation, shadows and screen-only controls.
- Tables use fixed layout and controlled row sizes to avoid page overflow.
- Images use fixed aspect-ratio frames and `object-fit: cover`.
- Image placeholders remain clearly labeled until approved photography is supplied.
